Componentes del Modelo Web

Paso 1

El Cliente (Navegador)

Es tu dispositivo (computadora o smartphone) ejecutando un navegador web como Chrome o Edge, encargado de formular las solicitudes.

Paso 2

Servidor DNS

La libreta de direcciones de Internet. Traduce los nombres de dominio amigables (como google.com) en direcciones IP numéricas.

Paso 3

El Servidor Web

Computadoras potentes que procesan tu solicitud, buscan los archivos HTML, CSS e imágenes requeridos y los envían de vuelta.
