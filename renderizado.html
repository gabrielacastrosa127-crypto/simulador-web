🔹 Paso 4: El servidor envía una respuesta HTTP

¿Qué ocurre cuando llega la respuesta?

El navegador recibe el código HTML enviado por el servidor y lo interpreta para mostrar la página web al usuario.

Etiquetas semánticas HTML5

  • header: encabezado principal
  • nav: menú de navegación
  • main: contenido principal
  • section: sección temática
  • article: contenido independiente
  • footer: pie de página

Ejemplo de código HTML

<header>
    <h1>Mi sitio web</h1>
</header>

<nav>
    Menú de navegación
</nav>
