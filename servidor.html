🔹 Paso 3: El servidor procesa la solicitud

¿Qué hace el servidor?

El servidor recibe la solicitud HTTP, procesa la información solicitada y prepara una respuesta para enviarla al navegador.

Ejemplo de respuesta HTTP

HTTP/1.1 200 OK
Content-Type: text/html
Server: Apache
  • Recibe solicitudes
  • Procesa datos
  • Envía respuestas
